# MAC Address vs IP Address

## What is a MAC Address?

A MAC (Media Access Control) Address is a unique physical (hardware) address assigned to every network device by the manufacturer. It helps identify a device within a local network (LAN).

**Example:**
00:1A:2B:3C:4D:5E

---

## What is an IP Address?

An IP (Internet Protocol) Address is a logical address assigned to a device on a network. It helps devices communicate with each other and allows data to reach the correct destination.

**Example:**
192.168.1.5

---

## Difference Between MAC Address and IP Address

| MAC Address | IP Address |
|-------------|------------|
| Physical (Hardware) Address | Logical (Network) Address |
| Assigned by the manufacturer | Assigned by the network/router |
| Usually permanent | Can change |
| Used inside a Local Area Network (LAN) | Used to communicate between different networks |
| Used by a Switch | Used by a Router |

---

## Summary

- MAC Address identifies a device.
- IP Address identifies the device's location on a network.
- Switches use MAC Addresses.
- Routers use IP Addresses.
